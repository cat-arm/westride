HTML & CSS Cheatsheet

HTML Basics

Name Meaning
Element Element = tag open + content + tag close e.g. <p>Hello</p>
Attribute <div class="main"> — class is attribute name, main is its value
Name attribute Defines property name e.g. width, height, position
Value attribute Defines value of the attribute e.g. 120px, left
Comment <!-- content --> is used for commenting in HTML
Link CSS file <link rel="stylesheet" href="style.css"> placed in <head>
Element[attribute] CSS selector for element with specific attribute
Box Model Box = content → padding → border → margin
Property The style you want to change e.g. background-color
Property Value What you set the property to e.g. red
<link rel="icon" href="favicon.ico"> Specifies the icon shown in browser tab (favicon)

Units

Unit Meaning
px 1/96 inch ≈ 0.26 mm
rem Relative to root element font-size (default 16px)
em Relative to parent element font-size
RGB rgb(0,0,205,0.8) — last value is alpha (opacity)
Hexadecimal #970515 is a hex color code
&nbsp; Non-breaking space in HTML

Flexbox Basics

Concept Meaning
display: flex Makes an element a flex container
Item shrink If container space < items, items shrink
Item expand If container space > items, items keep size
Main-axis Default: horizontal (row), can be set
Cross-axis Perpendicular to main-axis

HTML Tags Reference

Tag Description
<!DOCTYPE html> Defines HTML version
<html> Root HTML tag
<head> Metadata and styles/scripts
<meta charset="utf-8"> Defines character encoding
<title> Defines page title (seen in browser tab)
<body> Main visible page content
<h1>-<h6> Header tags from largest to smallest
<p> Paragraph
<div> Block-level container
<span> Inline container
<br>, <hr> Line break / horizontal rule
<i>, <em> Italic text
<b>, <strong> Bold text
<mark> Highlight text
<small> Smaller text
<del> Strikethrough
<ins> Inserted text (underline)
<sub>, <sup> Subscript / Superscript
<sup> Displays superscript text (above baseline)
<a>content</a> Defines a hyperlink
href="https://www.google.com" Specifies the link target (e.g. Google)
target="_blank" Opens link in a new tab
<img /> Embeds an image into the page

<style> Used to write CSS inside <head>
<section> Semantic section in HTML5

Table Elements

Tag or Attribute Description
<table> Defines a table structure
align="center" Aligns table/image content
<thead> Groups the header content in a table
<tbody> Groups the body content in a table
<tfoot> Groups the footer content in a table
<tr> Defines a row in a table

<td> Defines a standard data cell in a table
colspan="a" Merges a number of columns in a table
rowspan="a" Merges a number of rows in a table

Form Elements

Tag or Attribute Description

<ul> Defines an unordered list (e.g. bullet points)

<form> Container for form controls

<type="text"> Text field
name="content" Name used for the input field in submission
type="number" Numeric input
type="range" Slider input

type="radio" Radio button (select one only)
type="checkbox" Checkbox (multiple selections)
type="date" Date picker
type="password" Hides input with asterisks
type="reset" Resets all form fields
type="submit" Submits the form
<select> Creates a dropdown menu
<option> Option inside a <select>
<datalist> Defines suggestions for input
<list="id"> Associates with tag input with datalist
<textarea> Multiline text input
<button> Defines a clickable button
form="formId" Associates input/button with specific form

Responsive Design with Media Queries

Name Meaning
@media Apply styles depending on device screen conditions
@media (max-width: 600px) { body { font-size: 14px; } }
@media (width: 500px) Exact match condition
@media (width: 500px) { h1 { color: green; } }
@media (min-width: 768px) Apply if viewport width ≥ 768px
@media (min-width: 768px) { .box { padding: 2rem; } }
@media (max-width: 480px) Apply if viewport width ≤ 480px
@media (max-width: 480px) { nav { display: none; } }
@media (min-width: 400px) and (max-width: 800px) Apply for mid-range screen sizes
@media (min-width: 400px) and (max-width: 800px) { p { color: orange; } }
@media (orientation: landscape) Apply when screen is rotated horizontally
@media (orientation: landscape) { body { background: lightblue; } }

Flexbox

Name Meaning
display: flex; Make container a flexbox, enables all child elements to use flex behavior
.container { display: flex; }
display: inline-flex; Like flex but stays inline (does not break line)
.container { display: inline-flex; }
flex-direction: Defines direction of children
row | column | row-reverse | column-reverse
flex-direction: row;
justify-content: Horizontal alignment along main axis
flex-start | flex-end | center | space-between | space-around | space-evenly
justify-content: space-between;
align-items: Vertical alignment along cross axis (for single row)
stretch | flex-start | center | flex-end | baseline
align-items: center;
align-content: Vertical alignment of multi-line (wrap) rows
stretch | flex-start | center | flex-end | space-between | space-around
align-content: space-around;
align-self: Override vertical align for specific item
align-self: flex-end;
flex-wrap: Allow wrapping items when overflow
nowrap (default) | wrap | wrap-reverse
flex-wrap: wrap;
flex-grow: Ratio to grow extra space
flex-grow: 2; /* grows twice as fast */
flex-shrink: Ratio to shrink space if overflow
flex-shrink: 0; /* no shrink */
flex-basis: Initial main size before grow/shrink
flex-basis: 200px;
flex: Shorthand: grow shrink basis
flex: 1 1 200px;
order: Change visual order (default = 0)
order: -1;
flex-flow: Shorthand: direction + wrap
flex-flow: row wrap;
