HTML Cheatsheet

HTML Basics

Name Meaning
Element Element = tag open + content + tag close e.g. <p>Hello</p>
Attribute <div class="main"> — class is attribute name, main is its value
Name attribute Defines property name e.g. width, height, position
Value attribute Defines value of the attribute e.g. 120px, left
Comment <!-- content --> is used for commenting in HTML
Link CSS file <link rel="stylesheet" href="style.css"> placed in <head>
Element[attribute] CSS selector for element with specific attribute
Box Model Box = content → padding → border → margin
Property The style you want to change e.g. background-color
Property Value What you set the property to e.g. red
<link rel="icon" href="favicon.ico"> Specifies the icon shown in browser tab (favicon)
margin-inline-start Margin on the start side (left in LTR, right in RTL)
margin-inline-start: 1rem;
padding-block Padding on both top and bottom sides
padding-block: 2rem;

Units

Unit Meaning
px 1/96 inch ≈ 0.26 mm
rem Relative to root element font-size (default 16px)
em Relative to parent element font-size
RGB rgb(0,0,205,0.8) — last value is alpha (opacity)
Hexadecimal #970515 is a hex color code
&nbsp; Non-breaking space in HTML

Flexbox Basics

Concept Meaning
display: flex Makes an element a flex container
Item shrink If container space < items, items shrink
Item expand If container space > items, items keep size
Main-axis Default: horizontal (row), can be set
Cross-axis Perpendicular to main-axis

Responsive Design with Media Queries

Name Meaning
@media Apply styles depending on device screen conditions
@media (max-width: 600px) { body { font-size: 14px; } }
@media (width: 500px) Exact match condition
@media (width: 500px) { h1 { color: green; } }
@media (min-width: 768px) Apply if viewport width ≥ 768px
@media (min-width: 768px) { .box { padding: 2rem; } }
@media (max-width: 480px) Apply if viewport width ≤ 480px
@media (max-width: 480px) { nav { display: none; } }
@media (min-width: 400px) and (max-width: 800px) Apply for mid-range screen sizes
@media (min-width: 400px) and (max-width: 800px) { p { color: orange; } }
@media (orientation: landscape) Apply when screen is rotated horizontally
@media (orientation: landscape) { body { background: lightblue; } }
@media (prefers-color-scheme: dark) Applies styles when user prefers dark theme
@media (prefers-color-scheme: dark) { body { background: #111; color: #fff; } }
clamp(min, preferred, max) Scales font-size within a range
font-size: clamp(1rem, 2vw, 1.5rem);

Grid

Name Meaning
Grid container The element with display: grid; that holds all grid items
Grid cell Each individual item in the grid layout
Grid column / row Vertical or horizontal lines of cells; e.g., 3 columns × 4 rows = 12 cells
Grid line The boundary line between rows and columns, numbered from 1
Grid area A rectangular block covering multiple grid cells; e.g., 2x2 spans 4 cells
Gap / Gutter Spacing between rows or columns; set via gap: 10px;
Justify vs Align Justify aligns along horizontal (row/X axis), Align aligns along vertical (column/Y axis)
Line Reference Grid lines can be referenced by number:
R1\C1 C2 C3 C4
R2
R3
You can inspect lines in DevTools → Layout → Show line numbers.

Selectors Cheatsheet

Tag Selectors

Selector Description
div Selects all <div> elements
h1, h2, p Selects all <h1>, <h2>, and <p> elements
* Selects all elements

ID & Class Selectors

Selector Description
#id Select element with id="id"
.classname Select elements with class="classname"
tag.class Select tag with specific class
#id.class Select element with both id and class

Combinators

Selector Description
A B Select all B inside A (descendant)
A > B Select B that is direct child of A
A + B Select B that directly follows A (adjacent sibling)
A ~ B Select all B siblings after A (general sibling)
A, B Select all A and B elements

Attribute Selectors

Selector Description
[attribute] Has given attribute
a[attribute] Selects <A> elements that have the given attribute
[attribute="value"] Exact value match
[attribute^="value"] Starts with value
[attribute$="value"] Ends with value
[attribute*="value"] Contains value
